How it works

  1. Scheduler polls the mailbox inbox every poll interval seconds.
  2. For each unread message with PDF / DOCX / XLSX / PNG / JPG attachments, files are saved to a new case folder.
  3. Filenames are classified into Bill of Lading / Commercial Invoice / Packing List / iAuditor Report slots.
  4. The same extraction + report generator used for manual uploads runs against the case.
  5. The generated .docx report is emailed back to the sender, and the source message is marked as read.
